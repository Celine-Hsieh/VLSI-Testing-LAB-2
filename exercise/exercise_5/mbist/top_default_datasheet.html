Description:

The memory BIST (Built-In-Self-Test), or MBIST, is a technology-independent, synthesizable core for memory testing, specifically designed for highly flexibility and scalability. It features programmability of the test algorithms for various test requirement, and supports robust built-in test algorithms for comprehensive fault models, such as stuck-at, stuck-open, transition, address decoder and coupling faults, including bit-oriented and word-oriented ones.

When testing multiple memory cores in parallel, the MBIST can manage the test scheduling and test partition to achieve the optimized test time and area overhead.

Features:

  • Low cost memory BIST core for SRAMs
  • Support all popular memory architectures, including
    • synchronous/asynchronous memories
    • single-port/dual-port/multiple-port SRAM
    • two-port/n-read-m-write register files
    • read only memory (ROM)
    • flexible automation engine for memory BIST generation
  • Compliant to various memory compilers
  • Easily adapted for custom memory architectures
  • Rapid development cycle for widespread memory configuration
  • Support parallel testing for multiple memory cores
  • Allow test partition and test scheduling
  • Synthesizable to target logic process
  • Support industry standard memory test algorithms
  • Field programmable for user-defined test algorithms
  • Built-in robust test algorithm for comprehensive fault models of 100% coverage
  • Scan self-testable with high fault coverage (>99%)
